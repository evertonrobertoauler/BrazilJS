# Ferramentas para Analise do código fonte e também testes.

## Instalando as dependências


	$ npm install


## Servir a aplicação


	$ npm run server


## Rodar o Karma


	$ npm run karma


## Rodar o Protractor


	$ npm run protractor


## Rodar o karma e o Protractor


	$ npm run test


ou


	$ npm test
